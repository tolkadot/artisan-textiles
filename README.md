##This is a copy of Understrap with my personal modifications for my personal use.
1) Portfolio CPT
2) Slick Slider & styles
3) Social Shares Widget & Shortcode
4) Services Widget - to be used as a widget template
5) Navbar Logo/H1 stick and change size on scroll - cutsom-javascript.js
6) I run git from the dist folder and sync this with live site.
Updated the gulpfile so that the dist dir is not emptied when gulp dist is run.
7) Theme is intended to be used with ACF Pro
8) Custom styles for CF7

## Basic Features

- Combines Underscore’s PHP/JS files and Bootstrap’s HTML/CSS/JS.
- Comes with Bootstrap (v4) Sass source files and additional .scss files. Nicely sorted and ready to add your own variables and customize the Bootstrap variables.
- Uses a single minified CSS file for all the basic stuff.
- [Font Awesome](http://fortawesome.github.io/Font-Awesome/) integration (v4.7.0)
- Jetpack ready.
- WooCommerce support.
- Contact Form 7 support.
- [Child Theme](https://github.com/holger1411/understrap-child) ready.
- Translation ready.
